Peq Database Download

thj_peq = this is the peq you will need for the server. SQL command to run to install it
-mysql -u root -p thj_peq < thj_peq.sql
-Set Login and Password to root if you want it to work out of the box. If not you need to change password in 
-eqemu_config.json, login.json, quests\migrate_bags.pl, quests\migrate_items.pl




Client Download

download the Multiclass-Server Client Files on the github release or using this link below
https://github.com/Kraqurjak/Multiclass-Server-Release/releases/tag/V1.0

Download the RoF2 client from steam or some other source.
Uncompress the Multiclass-Server Client Files and drop them in the RoF2 folder.
Overwrite all files.

That's it, your client should now be compatible.
